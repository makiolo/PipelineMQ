# How compile
```bash
git clone https://github.com/makiolo/PipelineMQ.git
cd PipelineMQ
bash <(curl -s https://raw.githubusercontent.com/makiolo/cmaki/master/bootstrap.sh)
./cmaki/setup.sh
./cmaki/compile.sh
./cmaki/tests.sh
```
